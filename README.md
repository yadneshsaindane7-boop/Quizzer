# Quizzer App By Yadnesh Saindane

An interactive quiz web application built with HTML, CSS, and JavaScript to provide users with a fun and engaging way to test their knowledge.

## Live Demo

Try the live app here: [Quizzer App](https://quizzer-phi-pearl.vercel.app/)

## Features

- Multiple-choice quiz interface
- Real-time score tracking
- Interactive questions and options
- Responsive design for all device sizes
- Clean and modern user interface
- Immediate feedback on answers
- Navigation controls for progressing through questions
- Final score summary displayed at the end
